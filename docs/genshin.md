# Istaroth / Laimelea — 原神リファレンス

アプリ名「Laimelea」の元ネタとなった原神のキャラクター・設定の情報をまとめる。

## 1. イスタロト（Istaroth）とは

原神における**時の支配者**。天理（Primordial One / Phanes）が創造した**四つの影（Four Shades）**の一柱。四つの影はそれぞれテイワットの根本法則を司る：

| 影           | 司る領域 |
| ------------ | -------- |
| Naberius     | 生       |
| Ronova       | 死       |
| **Istaroth** | **時**   |
| Asmoday      | 空間     |

かつてモンド、淵下宮（エンカノミヤ）、Nod-Krai で広く崇拝されていたが、現在ではその存在自体がほぼ忘れ去られている。

## 2. Laimelea の名前の由来

「Laimelea」は Nod-Krai の **Frostmoon Scions**（霜月の末裔）がイスタロトを呼ぶ際の名前。

バルト神話の運命の女神 **Laima** に由来する：

- リトアニア語で「laime」= 幸福・運命・幸運
- Laima はリトアニア・ラトビア伝統における**運命・宿命・幸運の女神**
- ラトビア神話では Karta, Dekla と共に**運命の三女神**を構成（ノルンやモイライに相当）
- 「Laimelea」の綴りは 1666 年の Wilhelm Martini によるラテン語序文に初出

時の女神（イスタロト）と運命の女神（Laima）の接続は意図的：イスタロトは「瞬間」（運命の構成要素）を支配し、Laima は「運命そのもの」（瞬間の集積）を支配する。

## 4. 時・瞬間・風の三位一体

イスタロトは**時間・瞬間・風**の三つの概念を一体として体現する：

- **時間（Chronos）**: テイワットにおける時の流れそのものを支配
- **瞬間（Kairos）**: 「刹那の神」。連続的な時間ではなく、決定的な一瞬を司る
- **風（The Undying Wind）**: 風は時の比喩 — 目に見えず、常に存在し、常に前進し、触れた場所の記憶を運ぶ。**風は時を形にしたもの**
- **記憶**: イスタロトは存在の全てを記録する。「世界の物語を書き始めた」

## 5. ヴェンティ / バルバトスとの関係

- ヴェンティはかつて「千の風」の一つである微小な風の精霊だった。イスタロトこそが「千の風の時」そのもの
- 千風の神殿でイスタロトと出会い、**時の力と権限の断片を託された**
- バルバトスはイスタロトから授かった時の力で、マレ・ジヴァリの災厄を「時間軸から吹き飛ばした」
- 全言語で同じ声優がキャスティング（深い精神的つながりの示唆）
- 初期モンドでは両者が共に崇拝されていたが、ローレンス一族がイスタロト信仰を弾圧

## 6. ビジュアル・デザイン要素

### 公式ビジュアル（「テイワット章間奏テーゼ：神の限界」で公開）

- 長身・華奢な女性的姿
- **白い長髪** — 毛先が消え去るように薄れる（時の経過と消散の暗喩）
- 右側に編み込み（構造的要素と消散の対比）
- **金色の時計のような虹彩**（針のない時計 — 目盛りのみで針がない。測定を超越した時間）
- 白い短いドレス（背面が非対称に長い、流動感）
- 取り外し式の白い袖（先端が青にグラデーション）
- 精緻な金の胸飾り
- ペリウィンクル色のショートグローブ
- 金の足首飾り
- 裸足（超越的・彼岸的な存在感）
- **頭上に浮遊する2つの巨大な金の環** — 時計の装飾がぶら下がる（オーレリー / 時計機構のシルエット）
  - 5つの小さな時計の針がハロー周囲を周回（動的・回転の印象）
  - 左手にも光る大きなハローリングをアクセサリーとして装着
  - 髪にも1本の時計の針がヘアピンのように飾られる
- **風鈴のような吊り下がる金の棒**（時と風のモチーフの融合）
- **トリケトラ（三つ組結び目）** — 四つの影全員に共通するシンボル。標準的なケルト式トリケトラ（3つの交差する弧 / vesica piscis ×3）。テイワットでは正位置と逆位置の2種類が存在
- 星のモチーフ
- ゲーム内のパイモンメニューの時間スキップUIとハローデザインが酷似（意図的な一致）

### カラーパレット

| 色                          | 用途                                   | Hex（近似）           |
| --------------------------- | -------------------------------------- | --------------------- |
| 白                          | 髪・ドレス・主要色                     | `#FFFFFF` / `#F0F0F5` |
| 金 / ウォームゴールド       | 目・胸飾り・足首飾り・浮遊環・時計装飾 | `#D4A843` / `#C9A84C` |
| ペールブルー / アイスブルー | 袖のグラデーション                     | `#A8C8E8` / `#B8D4F0` |
| ペリウィンクル              | グローブ・アクセントカラー             | `#8B8FC7` / `#9999CC` |

全体的に**冷色調にウォームゴールドのアクセント**。白と淡い青が支配的で、金が神聖さ・天上的な温かみを添える。

## 7. 四つの影の目のデザイン対比

各影は固有の虹彩デザインを持つ：

| 影                 | 目のデザイン                                                                        |
| ------------------ | ----------------------------------------------------------------------------------- |
| Naberius（生）     | XY染色体パターン                                                                    |
| **Istaroth（時）** | **針のない時計盤** — 円形構造 + 目盛り + 同心円。針は目ではなくハローと髪飾りに配置 |
| Ronova（死）       | 目のモチーフが衣装全体に分散                                                        |

## 8. Nod-Krai / Frostmoon Scions のビジュアル

Laimelea の名を持つ Frostmoon Scions の視覚要素：

- **三日月**: 中心的シンボル。光るカギ型デバイスの頂部に三日月が配置
- **鹿角 / 角冠**: メンバーが鹿角のアクセサリーを装着。霜月の女神が命の最後に贈った角冠が純血の証
- **カラーパレット**: 寒色モノクローム — 白、銀、ダークブルー。永遠の冬
- **月 + 鹿角の複合紋章**: ローマの月の女神ディアナを彷彿
- 「新月の紡ぐ銀糸」「氷河の輝き」という語彙
- 一部建築は淵下宮の古代ギリシャ風と類似

月のサイクル（月齢）= 時間の計測手段 → アプリの「カスタムデイサイクル」コンセプトとの主題的重なり。

## 9. 千風の神殿のモチーフ

- モンド北東部の古代円形劇場の廃墟
- **日時計の碑文**（千風の神殿と名もなき島の両方に存在）：
  - 「風は物語の種を運び、時がそれを根付かせる」
  - 「風に運ばれた物語は、やがて伝説として花開く」
- 風（バルバトス）と時（イスタロト）の不可分性を刻む
- 石柱に「長い間忘れられた記念のシンボル」が微かに残る

## 10. 風渦 / 風元素のビジュアル言語

- **風元素シンボル**: 4つの葉が風車のように湾曲した回転モチーフ。ダイヤモンド形の輪郭内で渦を描く
- カラー: ティール / グリーン（`#74C2A8`）+ 白
- 風（目に見えない時の流れ）→ 渦巻き（可視化された時の動き）という変換が設計上のキー

## 11. 千の風と不滅の風

- **不滅の風（The Undying Wind）**: イスタロトの永遠の本質 — 死すべきものが朽ちる中、風（時）は決して止まない
- **千の風（The Thousand Winds）**: 時の流れを構成する無数の個々の瞬間の比喩。各瞬間が一つの風であり、それらが合わさって時の奔流を成す
- 「時の風」という慣用句は現代モンドの日常会話にも残っている（イスタロト自身は忘れられたにもかかわらず）

## 12. 採用されたアイコンデザイン — "Thousand Winds"（千の風）

"Thousand Winds"（千の風）はイスタロトの称号「千の風の時（The Thousand Winds of Time）」に由来するアイコンコンセプトである。時計アプリのアイコンとしての機能性を保ちながら、イスタロトへのオマージュを随所に織り込んでいる。

### デザイン構成

- **深い紺青の背景**（`#122035`）— Nod-Krai の永遠の冬の夜空を想起させる
- **金色の時計リング**（`#D4A843`）— イスタロトの頭上に浮遊する金の環を参照。**針がない**（イスタロトの虹彩 = 針のない時計盤）
- **12の目盛り** — 4つの主目盛り（12, 3, 6, 9）と8つの副目盛り
- **3本の青いトリスケリオン風渦**（`#5B9BD5`）— 時計の針の代わりに風が吹く。「風は時を形にしたもの」というイスタロトの本質を体現する。トリスケリオン（三脚巴）の3つの腕はトリケトラ（四つの影の共通シンボル）も暗示する
- **金の中心ドット** — 「瞬間（カイロス）」の象徴。全ての風が収束する一点

### イスタロトとの対応

| アイコン要素       | イスタロト参照                   |
| ------------------ | -------------------------------- |
| 針のない時計盤     | イスタロトの虹彩（針のない時計） |
| 金の環             | 頭上に浮遊する2つの金の環        |
| 青いトリスケリオン | 千の風 / 不滅の風                |
| 3本の腕            | トリケトラ（三つ組結び目）       |
| 紺青の背景         | Nod-Krai の永遠の冬              |
| 中心のドット       | カイロス（決定的瞬間）           |
